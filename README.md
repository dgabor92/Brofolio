# D&D Coaching

Modernized React landing page for Deutsch Tamás coaching practice.

## Tech stack

- Vite
- React 18
- Chakra UI
- Framer Motion
- Vercel ready static deployment

## Scripts

- `npm run dev` - start the Vite dev server
- `npm run build` - build for production into `dist/`
- `npm run preview` - preview the production build locally

## Deployment

The project is configured for Vercel with an SPA rewrite so deep links continue to resolve to `index.html`.
